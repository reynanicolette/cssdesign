What are some stylesheet selectors?

Selectors

  • HTML Elements

    body { background: black; };

  • Classes

    .blueText { color: blue; }

  • IDs

    #section { display: flex; }

  • :pseudoclass

    a:hover { transition: 0.3s all; }

  • ::pseudoselectors

    p::before { content: "Hey"; }
  • * (selects all elements)

Combinators

  • div.red
  • div p
  • a, a:hover
  • section + section
  • div > p
  • p ~ ul
