Grids

It’s time to set up your grid in your CSS. Doing this ahead of time saves you later when you’re attaching classes to your HTML elements. This is a new concept, and very helpful, so take notes!

You can name the columns in CSS based on the amount of columns they take up in the design.

So a column that takes up 50% of the full width of the container on your web page is 6 columns in your design.

.col-3 {
	width: 25%;
}

.col-4 {
	width: 33.33333%;
}

.col-6 { 	
    width: 50%;
}

.col-8 {
	width: 66.666666%;
}

.col-9 {
	width: 75%;
}

.col-12 {
	width: 100%;
}

Alternatively, you can name your columns based on the fraction of the container.

.quarter-width {
	width: 25%
}

.third-width {
	width: 33.33333%
}

.half-width {
    
    width: 50%;
}
.two-thirds-width {
	width: 66.666666%;
}

.three-quarters-width {
	width: 75%;
}

.full-width {
	width: 100%;
}

Either way, these are the basic columns you might use in your designs in the future!

Now you can style your headings.

What do they have in common? What makes them different?

This is where you can determine whether you should declare the HTML or create a class.

There are three headings in this document, but a document may have up to 6.

Since there’s not a lot in common between these items, you’ll have to call to them individually

I will use rems and shorthand to make this process faster.

h1 { 
    font: 600 4rem/4.5rem “Montserrat”;
}
    
h2 {
    font: 500 2rem/2.5rem “Montserrat”;
}
    
h3 
    font: 400 1.5rem/2rem “Montserrat”;
    letter-spacing: .25rem;
}

Note: In order for the font shorthand to work, you have to have the font name, which is a little redundant, but still more efficient nonetheless!

In print, designers refer to the space between the letters tracking. In CSS, it is referred to as letter-spacing. Check out this article to see how to use letter spacing. To see what units are possible in web in general, check this out. Since the letter spacing in the document is .25em, I will use that amount in rems.

Depending on your design, you might need to use a class to declare uppercase. In this comp, the uppercase is attached to two of our headings, but not all <*h1>, so we need to create a class. Do NOT forget to add this class to the proper elements in the HTML

.text-uppercase {
    text-transform: uppercase;
}

General Sections

To add enough spacing for the <*section>, it will be 5rem (equivalent of 80px) on the top and bottom.

section {
    padding: 5rem 0;
}

We can color the first two sections too using the :nth-of-type() pseudoclass. Instead of odd and even, we can use 1 & 2 for the first two sections.

We can apply a background image to the 3rd section as well.

This exercise is good practice for specificity. If this were a larger website, it might be a good idea to use an ID for these sections.

section:nth-of-type(1) {
    background: #E8E5D8;
}
section:nth-of-type(2) {
    background: #FFB190;
}
section:nth-of-type(3) {
    background: url("../images/physalis-bg.png") no-repeat top center;
    background-size: cover;
}

Section 1

In this section, we will use floats to create the columns.

We already added a clearfix class in our CSS, but we need to add the class in the HTML to the appropriate element. In this case, it’s the <*article>.

I created an ID of #floats and attached it to the <*article> so that I can use specificity to style the children (<*figure> and <*div>).

#floats-left figure, #floats-left div {
    float: left;
}
#floats-left div {
    padding: 4rem 0;
}

Section 2

In this section, we will use a new concept. Take your notepad out! It’s called flexbox. There are several resources to help you learn. We’re going to create horizontal columns.

I will apply an ID to the article called #flex.

#flex {
    display: flex; 
    align-items: center;
}
#flex h2 {
    margin-bottom: 1rem;
}

All of the default flex properties created horizontal columns.

We will use a concept here to style your content. It’s called CSS Grid.

Learn CSS Grid

Take the time to understand and practice if you need to.

First, we need to center the <*h1>

#grid div:first-of-type {
    text-align: center;
}

#grid div h1 {
    margin-bottom: 1rem;
}

#grid div:last-of-type {
    display: grid;

}

Note: These are very simple layouts, and as we move forward, you may use more properties for Flexbox or CSS Grid, so get to know these now while you code this exercise and project. Practice makes perfect! But without the practice, you might not understand these concepts completely.

This week you will fill the rest of the content on the page. Form items will still not need to be perfect, as we will cover them in next week's tutorials.

  • Use background images when appropriate. Remember, if there's anything that needs to be on top of an image, it should be handled like a background
